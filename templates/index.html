What is gold.css?

gold.css is a CSS stylesheet based on the golden ratio.

A work in progress.

Features

1

Ratios: everything is (or should be) based on the golden ratio.

Resets: Sensible CSS defaults so you can build and expand it easily.

2

3

Classes: Easy-to-use classes to make the most of flexboxes.
